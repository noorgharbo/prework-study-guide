HTML

Hyper Text Markup Language

  • HTML is the basic building blocks of websites
  • It is made up of elements to provide content and the basic framework of all pages on the web
  • !DOCTYPE is a declaration that tell the browser what type of document it should expect when it is opened.
  • HTML Attributes are always defined at the beginning. They define additional information about elements.

Elements

  • The head element contains information or metadata about the webpage.
    • Sample metadata tags include: title, style, base, link, meta, and script.
  • The body element contains the content that is shown to a user.
    • Elements contained inside body include:
      • Heading Element: H1 H2 H3 H4 H5 H6
      • Paragraph Element: P
      • List Element: ul ol li
      • Image Element: img
      • line breaks: br
      • Anchor (hyperlink) - a

CSS

Cascading Style Sheets

The Language used to describe the presentation of a website that was created with HTML and how each element on a webpage is rendered.

  • CSS describes the presentation of a website created with HTML
  • CSS describes how each element on a webpage is rendered
